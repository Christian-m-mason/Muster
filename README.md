# Muster

Muster is a comprehensive military management app designed to streamline Soldier data tracking. With automated notifications for admin due outs like DD93, SGLI 8286, and Azimuth Check (GAT), Muster ensures timely compliance. 

The app seamlessly integrates with DTMS, providing real-time updates. Additionally, Muster features an ACFT function enabling NCOs to create and grade tests effortlessly by scanning soldiersâ€™ QR codes, allowing efficient recording of times and scores for each event.
